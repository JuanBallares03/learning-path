# JavaScript Learning Journey

A progressive 18-day JavaScript curriculum covering fundamentals through advanced backend and security concepts. Each day includes hands-on exercises and a final project that applies the concepts in real-world scenarios.

---

## Technologies

**Frontend:** HTML5 · CSS3 · JavaScript (ES6+) · React 19 · Vite · JSX  
**Backend:** Node.js · Express.js · REST APIs · Fetch API · Axios  
**Database:** MongoDB · MySQL  
**Security:** bcrypt · AES-128-GCM · JWT · Regular Expressions  
**Tooling:** NPM · Webpack · ESLint

---

## How to Use

Each day is self-contained. Navigate to the folder of the day you're interested in and follow its README instructions.

```bash
# Example: run the React project (Day 14)
cd dia14-aplicaciones-web/07-proyecto-dia14/usuario-vs-maquina
npm install
npm run dev
```

> **Note:** Projects using Node.js (Days 15–18) require `npm install` before running. All `node_modules/` are excluded via `.gitignore`.

---

## Project Highlights

- **Days 1–7:** Foundations — HTML structure, CSS styling, JavaScript syntax, functions, control flow, loops, and DOM manipulation.
- **Days 8–12:** Intermediate — Objects, prototypes, ES6 classes, JSON data loading, and event-driven programming.
- **Days 13–14:** Asynchronous JS & Modern Frameworks — `async/await`, REST API consumption, and a first SPA with React.
- **Days 15–16:** Server Communication & Databases — Full CRUD with Fetch API and a REST API built with Express + MongoDB.
- **Days 17–18:** Security & Modular Architecture — Password hashing, JWT authentication, encryption, and ES modules with Webpack.
