# Day 3 – JavaScript Project: "Trivia with Countdown"

## 📌 Description
This project is a mini-game of 5 questions with a countdown timer.  
It represents the first contact with JavaScript, focusing on DOM manipulation, capturing user inputs, timers (`setInterval`, `setTimeout`), working with dates, and playing audio.

## ✨ Features
- 30-second countdown displayed and updated on screen.
- Game over sound when time runs out.
- Capture and display of user answers via `alert`.
- Display of submission date and time using `Date`.
- Reset button to restart the game.

## 🛠 Technologies
- HTML5  
- CSS3  
- JavaScript
