# Day 5 – JavaScript Project: "Movie Recommender"

## 📌 Description
This project is a movie recommender app that suggests a film based on genre and age.  
It focuses on decision structures in JavaScript, using logical operators, nested `if-else`, and `switch` statements to control program flow.

## ✨ Features
- 4 selectable genres: Comedy, Drama, Musical, Crime.
- 3 age ranges with different recommendations (12 total combinations).
- Age input validation.
- Result displayed directly on screen without page reload.

## 🛠 Technologies
- HTML5  
- CSS3  
- JavaScript
